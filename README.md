# 📦 Automação de cobranças para o sistema EVO

![Status](https://img.shields.io/badge/Status-Em%20Desenvolvimento-blue)

## 📖 Sobre o Projeto
Este projeto tem como objetivo otimizar cobranças manuais de um sistema de academia que utiliza a EVO. Em tempo real, o código exibe no output cobranças que deram certo ou não e qual o identificador (id) atual.

## 🚀 Tecnologias Utilizadas
- 🐍 Python
- ⚡ Selenium

## 🛠️ Como Configurar o Projeto
### 1️⃣ Clonar o Repositório
```bash
git clone https://github.com/mattospedrof/Automated_Collections.git
cd Automated_Collections
```
### 2️⃣ Criar e Ativar o Ambiente Virtual
```bash
python -m venv venv
source venv/bin/activate  # Sistema Linux/macOS
venv\Scripts\activate  # Windows
```
### 3️⃣ Instalar Dependências
```bash
pip install -r requirements.txt
```
### 4️⃣ Executar o Projeto
```bash
python main.py
```
* Necessário ter o Google Chrome instalado em seu computador.
